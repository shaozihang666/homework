# 论文导读：[Agentic Sketch Comedy Generation]
- 论文出处： [cs.AI]，2026年
## 研究背景与动机
人工智能在写作、编码、媒体生成等领域已展现出卓越能力，但「幽默创作」仍是一大挑战——当前大模型生成的笑话多是老套的双关语或“老爸笑话”，难以稳定产出能让观众发笑的内容。尽管通过精准提示词和多次迭代能偶现优质幽默内容，但要像专业喜剧工作室那样，可靠地生产出类似《周六夜现场》的短喜剧视频，仍需解决三大核心难题：
1. 构思合适的喜剧场景；
2. 创作真正有趣的剧本；
3. 生成高质量、一致性强且有吸引力的长视频（现有视频生成模型仅能产出短视频，且片段间一致性差）。

## 核心方法
本文提出了名为 COMIC（Content Optimization via Multi-agent Iterative Competition）的全自动化框架，核心设计如下：
1. **多智能体角色模拟**：模仿真人制作工作室的分工，设置编剧、评论家、导演等智能体角色，各角色各司其职；
2. **LLM 幽默评论家**：通过分析 YouTube 喜剧视频语料库及观众互动数据，训练与真人偏好对齐的 LLM 评论家，实现幽默效果的自动化评估；
3. **迭代竞争优化**：
   - 构建多个独立的“剧本岛”，每个岛由代表不同幽默风格的评论家委员会管控；
   - 剧本通过循环锦标赛竞争，落败剧本根据获胜者反馈迭代优化；
   - 写作循环（优化剧本）和渲染循环（优化视频）双链路迭代，保障内容广度与深度；
4. **全流程自动化**：输入仅需角色描述（文本、图像、声音）和背景参考，即可自动完成从剧本创作到多片段视频生成的全流程。

## 主要结果
实验表明，该框架产出的喜剧视频质量已接近专业制作的小品水平，同时在视频生成任务上达到了当前最优（state-of-the-art）性能，验证了：
1. 多智能体迭代竞争模式能有效提升喜剧内容的幽默性和多样性；
2. 基于真实观众偏好训练的 LLM 评论家，可可靠地完成幽默效果的自动化评估；
3. 模拟真人工作室的角色分工，能显著提升生成内容的完整性和专业性。

## 个人小结
COMIC 框架的核心创新在于将“多智能体竞争迭代”与“真人偏好对齐的幽默评估”结合，突破了大模型在幽默创作领域“偶发优质、整体拉胯”的瓶颈。该思路不仅适用于喜剧视频生成，也为其他主观评价类创作任务（如广告文案、创意写作）提供了参考方向。但目前仍有可优化空间：比如不同文化背景下的幽默偏好差异、长视频片段间的逻辑连贯性等，未来可进一步探索。
